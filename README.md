# Warwick-2026-Game-Jam
A First Person parkour game developed in UE5 with Blueprint, based on the given theme of 'No time to explain'. The game is set in an about-to-explode lab, with limited time for the player to escape. 

The game is downloadable through the following link: https://drive.google.com/drive/folders/1HEx4DrRyFG2mvWKiFIAHeXjfCEy1Jli9?usp=drive_link

Please download the whole folder named 'Windows'. Within the folder there is a file named 'GameJamgame.exe', running this file would run the game.
